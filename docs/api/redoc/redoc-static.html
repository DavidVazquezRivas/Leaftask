Users Management

Endpoints for managing users in the system, including listing and searching users

List All Users

Retrieves a paginated list of basic user profiles.

Allowed sort fields: firstName, fullName, email.

Authorizations:
BearerAuth
query Parameters
limit
integer [ 1 .. 100 ]
Default: 10

Maximum number of items to return.

cursor
string

The cursor token to fetch the next page of results. Leave empty to fetch the first page.

sort
Array of strings[ items^[a-zA-Z0-9_]+:(asc|desc)$ ]
Example: sort=firstName:asc

Multi-column sorting criteria. Format: field:direction (e.g., firstName:asc). You can pass multiple sort parameters to sort by multiple columns sequentially. Example: ?sort=firstName:asc&sort=email:desc

search
string

Partial search term to filter by fullName or email (e.g., "Joh" or "@example").

email
string <email>

Exact match filter for the user's email.
